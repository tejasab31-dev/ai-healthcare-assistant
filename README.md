# AI Healthcare Assistant

An AI-powered healthcare application that predicts diseases based on symptoms and provides AI-generated health recommendations.

## Features

* Disease prediction using Machine Learning
* FastAPI backend API
* Streamlit user interface
* OpenAI-powered health advice
* SQLite database for prediction history
* PDF report generation and download
* Prediction history dashboard

## Tech Stack

* Python
* FastAPI
* Streamlit
* Scikit-Learn
* OpenAI API
* SQLite
* ReportLab
* Git & GitHub

## Installation

```bash
pip install -r requirements.txt
```

Run Backend:

```bash
uvicorn mcp_server:app --reload
```

Run Frontend:

```bash
streamlit run app.py
```
